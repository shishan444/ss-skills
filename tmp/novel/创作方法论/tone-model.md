# 腔调设定模型

> 本文档是创作流程的一部分，适用于所有小说。每一部新小说除了worldbuilding.md之外，还要有一份tone-guide.md定义自己的腔调。worldbuilding.md管的是"世界长什么样"，tone-guide.md管的是"读起来什么感觉"。

---

## 核心原则

同一套技术规则（视角、字数、结构）可以写出完全不同质感的小说。腔调是小说区别于其他所有小说的指纹——不是"网文风格"四个字能概括的。

腔调不是附加的调味料，是从创意入口和世界观中长出来的。幽默类型、情感节奏、叙述声音，都跟主角的独特性、世界的规则、故事的内核强绑定。

---

## 九个维度

### 一、幽默发动机

笑点从哪里来？这是腔调的第一块基石。

每一部小说必须明确它的"笑点发动机"——不是"这部小说很搞笑"，而是"笑点具体由什么机制产生"。

三部分析中识别出的发动机类型：

| 发动机 | 机制 | 案例 |
|--------|------|------|
| 反差 | A碰上B，两者不兼容产生笑点 | 现代人思维碰上古代社会 |
| 荒诞 | 用极度认真的态度做极度不合理的事 | 用做菜技能对战仙人 |
| 自嘲 | 主角对自己的处境进行刻薄的内心评价 | "打人不疼的废物" |
| 误解 | 角色A以为发生了X，实际发生了Y，读者同时知道两边 | 古代人看主角行为觉得疯了 |
| 降维 | 用低维概念解释高维事件，或反过来 | 用MOBA术语描述仙侠大战 |
| 身体诚实 | 身体行为和内心意图矛盾 | 身体自动交出秘密，内心疯狂尖叫 |
| 尊严崩溃 | 强者被弱者的荒诞手段碾压后的心理反应 | 反派被迫跳小天鹅 |

一部小说可以用多个发动机，但必须有一个主调。主调决定了这部小说的"笑声性格"。

### 二、幽默类型组合

主调发动机 + 1-3个辅助发动机。

辅助发动机在特定场景激活——比如主调是"反差"，但在战斗场景激活"荒诞"，在情感场景激活"自嘲"。

每个发动机的触发条件必须明确：什么场景触发什么类型的笑点？

### 三、笑点密度与分布

笑点不是均匀分布的。必须定义密度曲线：

- 开头（前10章）：多密？需要多快抓住读者？
- 稳定期（10-50章）：密度怎么变？是保持还是调整？
- 深水期（50章+）：当剧情变重时，幽默怎么存活？
- 高潮章：笑点和情感重锤怎么交替？

核心原则：永远不让读者在任何一种情绪里停留太久。笑久了审美疲劳，严肃久了弃读。

### 四、叙述声音人格

叙述者是谁？这决定了全书的"听觉质感"。

可选的叙述者人格：

| 类型 | 特征 | 案例 |
|------|------|------|
| 主角内心延伸 | 第三人称但≈第一人称，叙述和内心独白无缝切换 | 逍遥小书生 |
| 双视角切换器 | 用语言风格区分不同视角角色 | 突然成仙了怎么办 |
| 解说员 | 冷静超然，在角色行动旁做点评 | 万界圆梦师 |
| 毒舌旁观者 | 对所有人（包括主角）都刻薄点评 | （待发掘） |
| 温柔回忆者 | 像在讲一个很久以前的故事，带着怀念 | （待发掘） |
| 诚实记录者 | 不点评不吐槽，只把荒诞的事如实记录（越如实越好笑） | （待发掘） |

叙述者人格一旦确定，全书的"声音"就定了。不能这章是解说员下章变成毒舌。

### 五、语言质感

用什么语言风格承载幽默？

需要定义：
- **词汇层次**：口语化/文学化/混合？网络梗的使用频率和方式？
- **句式偏好**：短句密集如机枪？长句绵延如河流？长短交替？
- **笑点收尾格式**：用什么收笑点？（极短句"贞操还在。" / 感叹号"卧槽！" / 话说到一半掐断"生活就像那什么" / 反问"这是遇到粉丝了？"）
- **视觉元素**：颜文字？表情描述？无？
- **内心独白的标记方式**：直接插入不标记？用"他心想"？用括号？
- **古代/现代语言混合规则**（如适用）：什么场景用什么语言？

### 六、情感节奏

幽默和情感怎么交替？

需要定义：
- **松紧模式**：连续多少章轻松后插入一章重锤？重锤后用多少章恢复？
- **情感重锤的位置**：Act结尾？关键转折？还是随机打击？
- **幽默在严肃场景中的存活方式**：严肃场景中幽默不停（用调侃化解沉重）？还是幽默暂时退场（让沉重充分发酵）再回来？
- **读者的体感曲线**：读完一章的预期体感是什么？（想笑/想哭/想骂人/想拍桌子/倒吸一口气）

### 七、氛围底色

表面是什么调性？底层是什么温度？

这个维度定义了小说的"情感锚点"——读者放下书之后记住的那种感觉。

| 表面调性 | 底层温度 | 案例感觉 |
|----------|----------|----------|
| 轻松搞笑 | 认真温暖 | 笑着笑着发现主角比正经人靠谱 |
| 荒诞爆笑 | 逞强孤独 | 笑着笑着发现他们其实害怕/孤独 |
| 荒诞正经 | 废物逆袭 | 笑着笑着发现被低估的东西最有价值 |
| 冷幽默 | 残酷真相 | 笑着笑着发现世界本来就是这样 |
| 热血燃 | 软肋温柔 | 打着打着发现强者最怕的是失去 |

底色不是"黑暗"或"光明"这么粗暴——它是读者笑完之后的余味。是苦的？暖的？空的？满的？

### 八、标志性手法

这部小说独有的、读者一看就知道"这是这部小说"的技法。

标志性手法必须满足两个条件：
1. **只在这部小说里出现**（或至少是这部小说的首创）
2. **反复出现**（不是偶一为之，是全书持续的技法）

三部分析中识别出的标志性手法：

| 手法 | 来源 | 描述 |
|------|------|------|
| 内心吐槽链 | 逍遥小书生 | 一个事件触发连续三四层内心吐槽，逐层荒诞，极短句收尾 |
| 古代人视角误解 | 逍遥小书生 | 古代角色用他们的认知框架解读现代行为，结论错误但逻辑自洽 |
| 降格收尾 | 逍遥小书生 | 严肃场景突然被日常细节打断 |
| 三层反差 | 突然成仙了怎么办 | 身体行为/内心尖叫/最终结果同时呈现 |
| 游戏术语降格 | 突然成仙了怎么办 | 用MOBA术语描述仙侠战争 |
| 同场景双视角 | 突然成仙了怎么办 | 同一事件从两个视角各写一遍 |
| 废物技能创造性使用 | 万界圆梦师 | 垃圾技能被发现了正经用途 |
| 反派尊严崩溃 | 万界圆梦师 | 强者被弱者的荒诞手段碾压后的心理反应 |
| 一本正经做荒诞事 | 万界圆梦师 | 极度专业的态度做极度不合理的事 |
| 解说员式旁白 | 万界圆梦师 | 叙述者突然插入一句人生点评 |

每部小说必须定义2-4个标志性手法。这些手法是这部小说的"签名"。

### 九、读者体感目标

读完一章，读者应该有什么身体反应？

- 想笑？哪种笑？（会心/爆笑/苦笑/倒吸一口凉气后的笑）
- 想哭？哪种哭？（感动/愤怒/心疼）
- 想骂人？骂谁？（主角/反派/作者）
- 想拍桌子？为什么？（爽/气/震撼）
- 倒吸一口气？因为什么？（反转/真相/格局打开）

体感目标不是单一的——不同章节有不同的体感目标。但全书的"体感基调"必须一致。比如基调是"会心笑"，那爆笑和苦笑可以有，但不能太多——否则基调会散。

## 与创作流程的整合

- **构思阶段（Phase 0）**：除了worldbuilding.md之外，完成tone-guide.md。九个维度全部定义。特别是"幽默发动机"和"标志性手法"——这两项是跟创意入口强绑定的。
- **写作阶段（Phase 1）**：每章写作前检查——这章的笑点发动机对不对？叙述声音人格一致吗？标志性手法出现了吗？每个Act的 `novel-stage-review` 中增加"腔调一致性检查"。
- **Review阶段**：`novel-stage-review` 的阅读体验、文风一致性和情绪路由兑现检查，用tone-guide.md作为评分标准。重点检查：笑点密度是否符合曲线？叙述声音有没有断裂？氛围底色有没有偏移？

## 三部案例对照表

| 维度 | 逍遥小书生 | 突然成仙了怎么办 | 万界圆梦师 |
|------|-----------|-----------------|-----------|
| 幽默发动机 | 反差（现代vs古代） | 双向反差+身体不由自主 | 废物技能降维打击 |
| 辅助发动机 | 自嘲/误解/降格收尾 | 游戏术语降格/尊严崩溃 | 甲乙博弈/解说员旁白 |
| 笑点密度 | 高→中高→中（递减但持续） | 全程极高 | 中高→极高（递增） |
| 叙述声音 | 主角内心延伸 | 双视角切换器 | 解说员 |
| 语言 | 口语化+偶尔网络梗 | 颜文字+密集网络梗+古文 | 正经叙述+荒诞内容 |
| 情感节奏 | 松紧交替，笑久了认真一下 | 极快切换，不在情绪里停留 | 任务制弧线，每个弧独立 |
| 氛围底色 | 表面轻松→底层温暖 | 表面爆笑→底层逞强孤独 | 表面荒诞→底层认真 |
| 标志手法 | 内心吐槽链/误解/降格收尾 | 三层反差/游戏降格/双视角 | 废物技能创造/尊严崩溃/正经做荒诞事 |
| 读者体感 | 会心笑+偶尔鼻酸 | 爆笑+偶尔心疼 | 笑着震惊+偶尔肃然起敬 |

## 一句话总结

worldbuilding.md定义了世界长什么样。tone-guide.md定义了读这个世界什么感觉。两者缺一不可，且必须从创意入口和主角核心设定中自然长出——不是套模板，是每部小说自己长出来的指纹。
